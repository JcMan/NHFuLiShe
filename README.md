#内涵福利社
本项目是内涵福利社的网页版，因为喜欢内涵福利社这款软件，所以就做了一个网页版的形式。

- 开发环境：Ubuntu14.04 Python2.7
- 开发工具：PyCharm 专业版 2016.2.3
- 开发框架：Django(1.6.1) Bootstrap

## 功能
本项目包括四个主要模块：

1. 段子
2. 趣图
3. 视频
4. 专辑
